1. Import the Plugin
2. Open the Demo Scene
3. In 'Socket Manager' Game Object configure the socket informations
4. In 'Socket Manager' 'Localized Text Key' attribute'key' is the same key that you need to configure in config.json external archive
5. Create in root build paste another folder with whatever name you want.
6. In 'Localization Manager' Game Object in attribute'Data Path' you paste the same name of the paste you've been create in step 5.
